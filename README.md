## Getting Started
### Prerequisites
* Node.js
* MongoDB
* React.js
### Installation(to run locally)
1. Clone this repository
```
git clone https://github.com/adrianpalomares/tweeter-client.git
```
2. Navigate to the directory
```
cd tweeter-client
```
3. Install the modules
```
npm install
```
4. Start the server
```
npm start
```
